Project - 1

CODE EDITOR
(HTML, CSS, JAVASCRIPT, REACT.JS)

ABSTRACT

The project “Web-Based Code Editor for Real-Time Programming” focuses on providing developers and learners with an interactive environment to write, edit, and preview code instantly. Built using React.js, the editor supports HTML, CSS, and JavaScript, allowing users to practice web development in a single platform. The system enables syntax highlighting, structured layouts, and seamless switching between coding panels. Real-time rendering ensures that changes in code are reflected immediately, enhancing the learning experience and debugging efficiency. The editor is lightweight, user-friendly, and designed for accessibility across devices. By combining simplicity with functionality, this project aims to serve as a practical tool for students, educators, and developers to improve coding productivity.

Project - 2

NEWS APP
(HTML, CSS, REACT.JS)

ABSTRACT

The project “Personalized News Application” is designed to deliver real-time news updates from multiple sources in a structured and user-friendly interface. Built with React.js and integrated with external news APIs, the system fetches and categorizes articles into sections such as politics, sports, technology, and entertainment. Users can search, filter, and bookmark news, ensuring personalized content delivery. The app includes a responsive layout for smooth access across devices and provides dynamic updates without manual refresh. Features like headline highlights, trending topics, and article previews enhance user engagement. By combining simplicity, interactivity, and speed, this project aims to provide an efficient news reading experience tailored to modern digital users
